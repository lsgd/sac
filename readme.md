# Print SAC sections sorted by membership fee
Prints the [SAC sections](https://sac-cas.ch) sorted by yearly membership fees ascending order.

The first number is the yearly membership fee, the number in parenthesis is the one-time fee for becoming a member of this section.

You can use the first argument to switch between the printed fees: youth, single, family

# Example

python3 sac.py family
